Defined in src/feat/math/LinearAlgebra.js:918

This routine decomposes an rows x cols matrix A, into a product of the three matrices U, W, and V', i.e. A = UWV', where U is an rows x rows matrix whose columns are orthogonal, W is a 1 x cols matrix, and V is an cols x cols orthogonal matrix.

Parameters:

  • A Matrix_t
  • W Matrix_t

    vector of singular values

  • U Matrix_t

    the left orthogonal matrix

  • V Matrix_t

    the right orthogonal matrix

  • [options] Number=0 optional

    Igata.SVD_U_T and/or Igata.SVD_V_T to return transposed U and/or V
